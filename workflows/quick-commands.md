# Quick Commands

Use these commands in chat after putting files into `inputs/`.

## Excel Only

```text
Use the Excel report workflow.
Input files are in inputs/.
Please clean the data, create summary tables, charts, and a final report workbook.
Save the final file to outputs/.
```

## PPT Only

```text
Use the PPT generation workflow.
Input files are in inputs/.
Use any PPT template in templates/ppt/ if available.
Please create a management-ready deck and save it to outputs/.
```

## Word Only

```text
Use the Word document workflow.
Input files are in inputs/.
Use any Word template in templates/word/ if available.
Please create a polished Word document and save it to outputs/.
```

## Excel + PPT

```text
Use the combined Office workflow.
Input files are in inputs/.
First create an Excel analysis workbook, then create a PPT based on the key findings.
Save both final files to outputs/.
```

## Excel + PPT + Word

```text
Use the combined Office workflow.
Input files are in inputs/.
Please create:
1. an Excel analysis workbook
2. a management PPT
3. a Word executive brief
Save all final files to outputs/.
```

## When Requirements Are Unclear

```text
Inspect the files in inputs/ first.
Then recommend the best Office automation output before creating the files.
```
