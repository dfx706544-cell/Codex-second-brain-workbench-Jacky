# 自动化工作台 v3 新手教程

这份教程教你使用新版工作台。你不需要会编程。

## 1. 打开工作台

优先运行这个脚本打开工作台：

```powershell
powershell -ExecutionPolicy Bypass -File "C:\Users\嘉十一\Documents\Codex\2026-06-24\w\automation-workbench\scripts\open-workbench.ps1"
```

它会自动打开类似这样的地址：

```text
http://127.0.0.1:8787/automation-workbench/app/
```

如果 8787 被占用，脚本会自动换成 8788 或后续端口。

打开后先看“执行队列”区域。如果显示“共享队列已连接”，说明你加入的任务 Codex 可以直接读取。

如果显示“本地备份模式”，说明当前页面还没有连上共享队列。请用上面的脚本重新打开工作台。

## 2. 最常用流程

1. 在“任务中枢”输入需求。
2. 选择主助手。
3. 勾选平台、能力和交付方式。
4. 点击“加入执行队列”。
5. 确认右侧提示为“共享队列已连接”。
6. 回到 Codex 聊天，发送：

```text
请打开我们制作的自动化工作台，处理里面的执行队列。
```

Codex 会读取这个文件：

```text
C:\Users\嘉十一\Documents\Codex\2026-06-24\w\automation-workbench\queue\tasks.json
```

## 2.1 如果你已经加入队列但 Codex 说没看到

这通常不是你操作错了，而是旧页面把队列存在浏览器自己的本地缓存里。

最快恢复方式：

1. 运行第 1 步里的 `open-workbench.ps1`。
2. 刷新你刚才加入任务的工作台页面。
3. 如果页面显示“共享队列已连接”，旧队列会自动同步出来。
4. 如果还是没有，请在新版页面里重新点一次“加入执行队列”。

## 3. 社交消息怎么处理

你可以这样说：

```text
请打开飞书、微信和 163 邮箱，帮我整理今天未回复的信息，按重要程度分类，先生成回复草稿。
```

流程是：

1. 我打开对应平台。
2. 如果需要登录，你在浏览器页面里亲自登录。
3. 我读取你授权页面中可见的消息。
4. 我生成回复草稿和跟进清单。
5. 你确认后，才可以继续发送。

不要把账号密码发到聊天框。

## 4. 邮件怎么处理

每日简报默认收件邮箱：

```text
jacky060911@163.com
```

默认模式是：

- 生成中文简报。
- 生成结构化报表或附件。
- 生成邮件正文草稿。
- 保存到 `outputs/`。
- 真正发送前等待你确认，或等你后续配置安全的邮件发送方式。

## 5. 账号数据复盘怎么用

你可以这样说：

```text
请读取我导出的达人沟通记录和抖音作品数据，分析回复率、成功率、流量、转化和变现，并给出明天的优化建议。
```

如果平台后台无法直接读取，你可以把数据导出到：

```text
inputs/
```

推荐字段：

- 达人沟通：触达数、回复数、有效回复数、意向数、成交数、话术版本、跟进次数。
- 作品数据：标题、发布时间、播放、曝光、完播、点赞、评论、收藏、分享、点击、成交、收入。

## 6. 自媒体/IP 助手怎么用

剪辑助手已经升级成“自媒体/IP 助手”。它可以帮你做：

- 个人 IP 定位
- 选题库
- 脚本
- 剪辑方案
- 标题封面建议
- 发布时间建议
- 复盘指标
- 变现路径

例如：

```text
请结合我最近 10 条抖音作品数据，帮我设计明天 3 条视频的选题、脚本和剪映剪辑方案。
```

## 7. 每天早上 8 点简报

简报会覆盖：

- 美股、港股、宏观和重大公司新闻
- 跨境电商、TikTok Shop、Kalodata、FastMoss
- TikTok、YouTube、Instagram、抖音、微信视频号、小红书
- 自媒体/IP、平台算法、创作者玩法和变现机会
- AI、科技和学术研究
- 重要社会热点

要求：

- 中文通俗解读。
- 中外信息源都要覆盖。
- 每条重要信息都带真实网址。
- 不编造数据和来源。

## 8. Skill Scout 自我迭代

你授权我去 GitHub 找好用的 skills，但安全流程仍然是：

1. 我先搜索候选。
2. 我说明来源、用途、风险和维护状态。
3. 我告诉你会安装到哪里。
4. 你确认具体候选后，我再安装。

安装后需要重启 Codex 才能完整识别新 skill。

2026-06-25 已安装一批扩展 skills，包括网页自动化、Windows Office 桌面自动化、PDF/动效、邮件草稿、外部 app 连接和金融研究。工作台已经把它们加入能力列表。

其中 `connect` 是高级能力，可能连接 Gmail、Slack、GitHub、Notion 等外部平台并执行真实动作。它默认不会自动勾选；使用前你需要在当次任务里明确确认平台、账号、动作和内容。

## 9. 重要边界

我可以帮你打开平台、整理信息、生成草稿、生成文件、做复盘。

我不会直接做这些事，除非你在当次任务里明确确认：

- 发送微信/飞书/私信
- 发送邮件
- 上传附件
- 提交表单
- 安装第三方 skill
- 金融真实下单
- 输入密码、验证码、支付码或交易密码

## 10. 第二大脑 v4 怎么用

工作台现在多了几个栏目：

- 知识库：看收录的资料、来源、收录时间、发行时间、影响和下一步建议。
- 历史记录：回看通过工作台完成过的任务，方便按电商、金融、Office、系统等分类追溯。
- 每日交付：把每天早上的信息简报和业务反馈拆成两份邮件草稿。
- 个人画像：查看工作台对你的目标、偏好、约束和工作方式的理解。

现在也新增了三个助手：

- 个人成长助手：帮你沉淀心理学、逻辑学、经济金融、社交公关、书单和练习计划。
- 健康助手：帮你安排训练、饮食、作息和身材管理；涉及疾病、疼痛、药物或长高医学问题时，会建议咨询医生。
- 个人画像助手：只记录你确认过的偏好和目标，不记录密码、验证码、支付码、交易密码等敏感信息。

每天邮件会拆成两份草稿：一份“信息简报”，一份“业务反馈”。真正发送前仍然需要你确认。
