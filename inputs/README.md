# Inputs

Put source files for each automation job here.

Recommended naming:

```text
2026-06-sales-raw.xlsx
2026-06-customer-list.csv
2026-06-meeting-notes.txt
2026-06-reference-deck.pptx
```

For larger jobs, create one folder per task:

```text
inputs/monthly-review-2026-06/
```

Then put all related data, notes, and references inside that folder.
