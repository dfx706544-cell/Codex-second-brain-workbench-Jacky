# 每日金融业务与内容平台快讯

## 固定指令

```text
执行每日金融业务与内容平台快讯。
请用 AnySearch 和可用网页搜索查询最新信息，覆盖：
1. 美股和港股
2. 与股票涨跌强相关的新闻催化
3. 时政新闻和社会热点
4. 跨境电商、Kalodata、FastMoss 相关机会
5. AI 技术最新发展：模型、Agent、多模态、开源模型、AI 芯片、基础设施、监管、安全和可落地工具
6. TikTok、YouTube、Instagram、抖音、微信视频号、小红书的热点、平台玩法和算法/推荐机制变化

输出：
- 今日最重要 10 条消息
- 对美股/港股可能影响
- 对跨境电商业务可能影响
- 对内容创作者的可执行建议
- AI 技术最新发展及其对无垠工作台、剪辑、自媒体/IP、电商 BD 和投资观察的影响

- 运行与维护成本：同步 GitHub Actions、Codex/OpenAI/API、邮箱、第三方平台订阅和本地开机成本
- 来源链接
```
