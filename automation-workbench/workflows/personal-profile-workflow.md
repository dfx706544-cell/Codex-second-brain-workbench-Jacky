# 个人画像助手工作流

目标：让工作台逐步理解用户的目标、偏好、约束、工作方式和长期计划。

## 可记录内容

- 长期目标。
- 工作偏好。
- 常用平台。
- 输出格式偏好。
- 安全边界。
- 学习、事业、健康和个人成长目标。

## 不记录内容

- 密码、验证码、支付码、交易密码。
- 身份证、银行卡等敏感身份信息。
- 未经用户确认的敏感私人信息。
- 任何可用于绕过平台权限或安全验证的信息。

## 执行步骤

1. 从用户明确表达、已完成任务和用户确认过的偏好中提取候选画像。
2. 标注每条画像的来源、更新时间和用途。
3. 将候选变更先作为建议展示给用户确认。
4. 用户确认后，才建议写入 `automation-workbench/data/personal-profile.json`。
5. 用户要求修改或删除时，优先执行修改/删除建议，并保留变更说明。

## 输出

- 建议新增、修改或删除的画像条目。
- 为什么建议记录。
- 对工作台下一步自动化的影响。
- 用户可接受、修改或拒绝的清单。
