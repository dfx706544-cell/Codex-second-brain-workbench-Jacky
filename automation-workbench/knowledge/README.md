# 自动化工作台知识库

这个目录用于保存可追溯的市场、业务、平台和学术信息。任何更新都要保留来源链接，避免把未经验证的传闻写成结论。

## 文件分工

- `market-intel.md`：金融市场、宏观、跨境电商、AI 行业和业务机会。
- `academic-intel.md`：AI、营销、消费者行为、内容推荐、电商转化等学术信息。
- 后续可按主题新增文件，例如 `creator-platform-intel.md` 或 `beauty-category-intel.md`。

## 每条信息的推荐格式

```markdown
## YYYY-MM-DD 主题

- 来源：链接 1；链接 2
- 摘要：一句话说明发生了什么。
- 可信度：官方 / 主流媒体 / 平台公告 / 待核实
- 对我的影响：它和美股、港股、跨境电商 BD、内容创作或选品有什么关系。
- 下一步动作：我可以搜索、联系、整理、监控或避开的动作。
```

## 更新方式

在工作台点击“复制更新任务”，然后在 Codex 聊天里发送。Codex 会搜索最新信息，按上面的格式写入知识库或给出简报。
