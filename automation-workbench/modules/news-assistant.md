# 资讯助手

## 目标

定时获取对金融和业务有帮助的最新消息，并站在交易、跨境电商和内容创作者三个视角整理。

## 默认推送时间

```text
每天 09:00，Asia/Shanghai
```

之后可以改成：

- 美股盘前
- 港股开盘前
- 午间
- 晚间复盘

## 关注范围

### 金融

- 美股
- 港股
- 利率、通胀、就业、汇率
- 财报和业绩指引
- 分析师评级变化
- 政策和监管
- 地缘政治
- 大宗商品冲击

### 业务

- 跨境电商
- 外贸
- AI 行业
- 客户、竞品、供应链
- Kalodata 和 FastMoss 可辅助观察的 TikTok Shop 数据

### 内容平台

站在创作者视角，重点看：

- TikTok
- YouTube
- Instagram
- 抖音
- 微信视频号
- 小红书

关注内容：

- 最新热点
- 平台玩法
- 算法和推荐机制变化
- 新功能
- 热门内容格式
- 爆款开头和脚本结构
- 平台电商和转化玩法
- 创作者变现机会

## 输出格式

每次推送建议包含：

```text
1. 今日最重要金融消息
2. 可能影响美股/港股涨跌的催化
3. 业务和跨境电商机会
4. 内容平台热点和创作者玩法
5. 今天建议关注的行动清单
6. 来源链接
```

## 使用指令

```text
用 AnySearch 查询今天对金融和业务有益的最新消息。覆盖美股、港股、时政、社会热点、跨境电商、TikTok、YouTube、Instagram、抖音、微信视频号和小红书，并站在创作者视角总结平台玩法和算法变化。
```
