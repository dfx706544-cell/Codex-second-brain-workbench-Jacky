# 集成路线图

## 第一阶段：本地工作台

目标：把五个模块放进一个入口。

完成项：

- 本地网页工作台
- 配置文件
- Office 工作流
- AnySearch 资讯入口
- 平台和剪映入口
- 金融模块安全边界

## 第二阶段：固定任务模板

目标：让你一句话启动常见任务。

建议先做：

- 每日业务资讯
- 月度 Excel 报表
- 经营分析 PPT
- 询盘筛选表
- 短视频剪辑方案

## 第三阶段：平台连接

目标：接入真实平台。

需要你提供：

- 平台名称
- 登录方式
- 是否有 API
- 是否允许自动化访问
- 常用筛选规则
- 输出字段

## 第四阶段：交易监控

目标：先做非实盘。

内容：

- 行情源
- 新闻源
- 信号规则
- 纸面交易
- 风控仪表盘
- 复盘报表

## 第五阶段：人工确认式交易

只有在你确认平台、API、风控和合规后，才考虑这一步。

原则：

- 不能无人值守全自动实盘
- 必须有人工确认
- 必须有仓位限制
- 必须有亏损熔断
- 必须有完整日志
