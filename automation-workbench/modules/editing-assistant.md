# 剪辑助手

## 目标

打开剪映，并通过对话帮你完成剪辑规划和执行清单。

## 已配置剪映路径

```text
C:\Users\嘉十一\AppData\Local\JianyingPro\Apps\JianyingPro.exe
```

## 当前能力

- 打开剪映
- 生成剪辑方案
- 生成镜头顺序
- 生成字幕文案
- 规划转场和节奏
- 推荐 BGM 风格
- 生成短视频脚本
- 输出适合 TikTok、抖音、小红书、视频号的版本建议

## 使用指令

```text
启动剪辑助手。打开剪映，我要做一个 60 秒短视频，主题是新品推广。请给我镜头顺序、字幕文案、节奏、转场、BGM 和导出规格。
```

## 注意

剪映没有开放稳定通用的自动剪辑接口时，我会优先做“剪辑方案 + 草稿规划 + 打开软件 + 逐步指导”。如果后续能确认剪映草稿文件结构或可控接口，再升级为更深度的半自动剪辑。
