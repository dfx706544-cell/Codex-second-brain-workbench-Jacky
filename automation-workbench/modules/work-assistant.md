# 工作助手

## 目标

围绕跨境电商询盘、选品、达人、竞品和内容表现做筛选和跟进。

## 当前平台

- Kalodata: https://www.kalodata.com/
- FastMoss: https://www.fastmoss.com/
- 达秘 / TikClubs: https://www.tikclubs.com/workbench/function_introduction

## Kalodata 适合观察

- TikTok Shop 商品
- 达人
- 视频
- 直播
- 店铺和销量线索
- 市场趋势

## FastMoss 适合观察

- TikTok 电商数据
- 选品
- 竞品
- 达人
- 广告
- 直播和视频表现

## 达秘 / TikClubs 适合观察

- TikTok 网红达人自动营销
- 达人搜索和候选池补充
- 触达线索和沟通跟进
- KOL/KOC/UGC 合作线索
- BD 话术和跟进优先级
- 与 Kalodata、FastMoss 交叉验证达人质量

## 筛选维度

```text
商品热度：
增长速度：
价格区间：
达人质量：
内容爆发力：
直播转化：
竞品强度：
地区市场：
供应链可行性：
询盘质量：
跟进优先级：
```

## 使用指令

```text
启动工作助手。请打开 Kalodata、FastMoss 和达秘 / TikClubs，帮我筛选跨境电商机会，关注商品热度、达人表现、竞品、视频/直播表现、触达线索和询盘质量，并输出跟进清单。
```
